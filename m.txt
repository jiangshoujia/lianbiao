mkdir用于创建一个心得文件夹，git init用于创建一个新的仓库，touch用于在该仓库中创建新的文件
git中如何删除文件
1、rm -rf+文件名将文件转移到暂存区之中
2、git rm+文件名将文件从暂存区之中删除
3、删除之后需要提交修改的备注使用 git commit -m'备注信息'
将文件添加到仓库之中的操作：
1、git add 添加到暂存区
2、git commit 提交到仓库
git status用于查看状态
vim/vi用于修改其中的文件
修改完成之后按下esc 兵按下’：wq'实现保存并退出。每次修改之后必须向仓库之中进行提交
执行仓库的初始化代码之前需要先进入到仓库之中 使用cd+文件名然后再执行仓库的初始化。
使用git下载GitHub上面文件 git clone+仓库网址。
